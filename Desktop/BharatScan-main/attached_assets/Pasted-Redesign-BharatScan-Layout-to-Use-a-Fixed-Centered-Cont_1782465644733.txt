Redesign BharatScan Layout to Use a Fixed Centered Content Area on Every Page

Objective

Redesign the layout of the entire BharatScan website so that every page uses the same centered content layout as the Create Scan page.

The Create Scan page is currently the reference design.

Do not redesign the UI.

Do not change colors.

Do not change components.

Do not change spacing inside components.

Only change the overall page layout so that every page is centered within a fixed-width content container.

⸻

Current Problem

At present:

* Home page stretches across the full browser width.
* Scanner Dashboard stretches across the full width.
* Saved Scans stretches across the full width.
* Portfolio stretches across the full width.
* Alerts stretches across the full width.
* Settings stretches across the full width.

However,

The Create Scan page is perfectly centered.

It has equal blank space on both sides.

This is the desired layout.

I have attached screenshots showing the desired alignment using two white guide lines.

Everything inside those white lines is the website.

Everything outside those white lines should remain empty.

⸻

Design Goal

The website should have a fixed maximum content width.

The content should always remain horizontally centered.

Example

Browser Window

⸻

| Advertisement Area | Website Content | Advertisement Area |

⸻

The advertisement areas should remain empty.

Do not place any website components there.

⸻

Future Purpose

The blank space on both sides is intentionally reserved.

In the future, after publishing BharatScan, these areas will be used for:

• Google AdSense

• Banner advertisements

• Sponsored content

Therefore these margins must remain permanently available.

Do not allow website components to expand into these areas.

⸻

Create One Global Layout

Instead of individually centering every page,

Create one reusable layout component.

Example

AppLayout

MainLayout

DashboardLayout

ContentContainer

RootContainer

(or any suitable name)

Every page should automatically use this layout.

Avoid duplicating layout code.

⸻

Maximum Width

Use a fixed maximum width for the content.

Choose a width that closely matches the current Create Scan page.

Approximately:

1500px–1600px maximum width

The content should never become wider than this.

It should always remain centered.

⸻

Responsive Behaviour

Large Desktop

Website stays centered.

Blank margins remain on both sides.

Ultra-wide Monitor

Website does NOT stretch.

Blank margins simply become larger.

Laptop

Website remains centered.

Tablet

Container becomes fluid when necessary.

Mobile

Container should automatically become full width.

No horizontal scrolling.

Only the main content area should be centered.

Consistency

Every page must have identical left and right margins.

When navigating between pages,

The content should not shift horizontally.

Everything should remain perfectly aligned.

⸻

After this update:

✓ Every page has the same width.

✓ Every page is perfectly centered.

✓ Equal blank space exists on the left and right.

✓ The website looks consistent.

✓ Create Scan becomes the layout standard for the entire application.
