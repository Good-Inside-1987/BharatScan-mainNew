STEP 5 — Calculated fields formulas

For fields not already in the simulator, use these formulas:

Intrinsic Value (Spot): CE = max(0, spot - strike), PE = max(0, strike - spot)
Intrinsic Value (Futures): CE = max(0, futures - strike), PE = max(0, strike - futures). Use synthFutures price from the options data if available, else fall back to spot.
Time Value: LTP - Intrinsic Value (Spot). If negative, show 0.
Breakeven: CE = strike + CE_LTP, PE = strike - PE_LTP
Breakeven %: ((Breakeven - spot) / spot) × 100. For PE: ((spot - Breakeven) / spot) × 100
Per Strike PCR: Put OI / Call OI for each strike row. Show "∞" if Call OI is 0.
Probability of Profit (POP): For CE = (1 - Delta) × 100%, For PE = |Delta| × 100%. Round to nearest integer and show as percentage.
OI Change: current OI - previous bar OI. If previous data not available show "—".
OI Change %: (OI Change / previous OI) × 100. Show "—" if not available.
OI-lakh: OI / 100000 rounded to 1 decimal place.
LTP Change: current LTP - previous close LTP. Show "—" if not available.
LTP Change %: (LTP Change / previous LTP) × 100. Show "—" if not available.
IV Change: current IV - previous IV. Show "—" if not available.
Volume: total traded volume for the strike. Show "—" if not in data.
Bid / Offer: Show from data if available, else show "—".
