Fix Global Page Alignment Across the Entire BharatScan Website

Objective

The BharatScan website currently has inconsistent page alignment.

Every page uses a different content width and different horizontal positioning.

I have attached screenshots.

In every screenshot I have drawn two vertical white guide lines.

These white lines represent the permanent left and right boundaries of the website.

Every page must stay completely inside these boundaries.

No component should start before the left guide line.

No component should extend beyond the right guide line.

These two guide lines become the universal content boundaries for the entire application.

⸻

Current Problem

The permanent header already establishes the visual width of the application.

However, every page below the header ignores these boundaries.
